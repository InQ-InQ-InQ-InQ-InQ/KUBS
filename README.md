# 🎨&nbsp;&nbsp;&nbsp;What is KURS

## 🚀&nbsp;&nbsp;Goal
- 도서관을 제외한 교내 전 공간에 대한 대여서비스를 `한 웹사이트로 통합하여 구축`
- 교내 구성원만 가입할 수 있는 `학교 이메일 인증을 통한 회원가입` 서비스 구현
- 날짜별, 시간별 예약 데이터를 DB를 `MySQL과 연동`하여 저장
